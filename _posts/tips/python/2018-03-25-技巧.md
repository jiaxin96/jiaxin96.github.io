# Py的一些小技巧


## 字符串类

### 提取中文

```python
text = "hello你好"
p = re.compile(u'[^\u4e00-\u9fa5]')  # 中文的编码范围是：\u4e00到\u9fa5
text = "".join(p.split(text))
print(text)
```

> 输出：
> 你好
